Игра "Алхимик"

Статус. Выполнен 10.10.2017

Описание игры

Игра по мотивам культовой игры из 2000-х

Изначально игроку даны 4 элемента: вода, земля, огонь, воздух. Соединяя их, игрок получает новые элементы. Например:

  • земля + огонь = лава
  • воздух + огонь = энергия

Цель игры – открыть все элементы

Технологии, фреймворки и библиотеки, задействованные в проекте

  • Анимирование View без покадровой анимации
  • Single-Activity
  • Floating Action Button с дочерними анимированными элементами. Нажатие на парящую кнопку вызывает анимированное отображение еще трех кнопок. Повторное нажатие «сворачивает» элементы в исходное состояние.
  • Хранение пар ключ-значение в базе данных SQLite
  • Интегрирована реклама Google AdMobe
  • Реализовано получение вознаграждения за просмотр рекламного видео - технология AdMob Rewarded Video.
  • Для отображения всех списков используется RecyclerView

Game "Alchemy"

Project status. Completed 10.10.2017

c7bb54

Remake of famous game from 2000

Game description

Game based on the cult game from 2000. Initially, the player is given 4 elements: water, earth, fire, air. By connecting them, the player gets new elements. For example:

  • earth + fire = lava
  • air + fire = energy

Goal of game is to open all the elements

Technologies, frameworks and libraries used in the project

  • Animate View without frame-by-frame animation
  • Single-Activity
  • Floating Action Button with animated child elements. Clicking on the hovering button causes an animated display of three more buttons. Clicking again "collapses" the elements to their original state.
  • Storing key-value pairs in a SQLite database
  • Integrated Google AdMobe advertising
  • Implemented receiving a reward for watching an advertising video - AdMob Rewarded Video technology.
  • All lists are displayed using RecyclerView
