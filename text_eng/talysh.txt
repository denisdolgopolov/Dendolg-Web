# Талыши - кто?

Талыши - народ на Южном Кавказе. Талышский язык относится к иранской языковой группе индоевропейской семьи языков. По некоторым оценкам, общая численность талышей во всём мире составляет от 267 300 до 619 000 человек, а численность носителей талышского языка от 218 100 до 1 300 000 человек.

[Подробнее](https://ru.wikipedia.org/wiki/Талыши)

# История проекта

Официальное описание гласит: приложение «Talysh Dictionary (by F. Aboszoda)» является продуктом научно-просветительского проекта «TOLYSHNOMA», созданного Союзом талышской молодежи (гор. Москва) при содействии Талышской Национальной Академии (гор. Минск). Приложение создано при финансовой поддержке Правительства Талыш-Муганской Автономной Республики в эмиграции (гор. Гаага).

Хорошие люди занимаются спасением родного языка. Google Translater и другие крупные онлайн-переводчики не поддерживают талышский язык. Поэтому было решено создать собственное приложение, способное переводить с талышского на русский, английский, азербайджанский, персидский и обратно. 

# Исходники

В процессе создания были "отцифрованы" 4 словаря. Задача заключалась в создании простой, быстрой и чистой базы данных из pdf-файлов с переводами. 

Автором всех 4-х словарей является ученый Фахраддин Абосзода (Аббасов). В данном приложении предоставлены: 

* Русско-талышский словарь (60 000 слов; Мюнхен, 2012)
* Тюрко-талышский словарь (55 000 слов; Баку, 2015)
* Англо-талышский словарь (250 000 слов; Мюнхен, 2012)
* Персидско-талышский словарь (рукопись)

# Сложности

Главной сложностью стал персидский язык:

* начертание справа налево
* вязь - каждая следующая буква может изменить внешний вид предыдущей-
* наличие внешне одинаковых символов с разным значением в таблице кодировок. Например, два слова в HEX-виде: ﻣﺎﭼﮫﺳﮓ (fee3 fe8e fb7c fbab feb3 fb93) и ماچھسگ (645 627 686 6be 633 6af). На вид - одинаковые, но при сравнении не сойдется ни один символ, так как битовые значения различны.

# Технологии и решение проблем

* IText - для парсинга исходных PDF-документов в базу данных вида слова-перевод
* Kotlin - для написания алгоритма парсинга
* String Normolizer - для приведения разновидностей персидской вязи к единому стандарту (NFKC)
* Firebase Database - для хранения переводов, добавленных через Web-интерфейс
* Firebase API/SDK - для доступа к базе данных
* HTML/JS - для веб-интерфейса добавления новых переводов
