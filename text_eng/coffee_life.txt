Скидка на кофе за подписку

# Описание проекта

Агрегатор кофеен и ресторанов. Пользователь может приобрести подписку. Уровень подписки определяет процент скидки, которую пользователь будет получать на напитки и сопутствующие товары.
Для получения скидки, пользователю достаточно после получения заказа через приложение отсканировать QR-код на чеке.
Приложение позволяет найти лучшее заведение по нескольким критериям:


* месторасположение - все доступные точки отображаются на кликабельной карте
* рейтинг от пользователей - можно оставить комментарий с оценкой
* списк дополнительных удобств - наличие розеток, WI-FI, мягких диванов и т.д.
* цена на товары и услуги - весь ассортимент и информация о нем доступна в приложении

После регистрации пользователь может совершать заказы, копить баллы, вводить промокоды, ставить оценки и приглашать других пользователей, получая за это скидку на товары.

# Технологии, фреймворки и библиотеки, задействованные в проекте
    
* Retrofit 2.0 - общение с сервером
* Auth0 - авторизация через социальные сети (Facebook, VK, Google+)
* Yandex Map - для отображения местоположения кофеен и пользователя
* OkkHttp3 - для отправки HTTP запросов
* Cloud Payment - для привязки банковской карты к аккаунту пользователя, совершения оплаты товаров и оформления подписки
* Zxing - для сканирования и расшифровки QR-кодов
* Material Design - соответствие современным стандартам
* AndroidX - уменьшение веса приложения
* Recycler View - для списков
* Navigation Drawer View - для бокового меню
* Адаптивная верстка - соответствует дизайн-макету на всех телефонах и планшетах
* Язык приложения - Kotlin
* Минимальная версия - Android 4.4 (97% мирового рынка по данным Google)
