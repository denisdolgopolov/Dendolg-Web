Заказ такси
Статус. Выполнен 01.08.2020 - 14.08.2020
#80F9C34A
Верстка приложения для заказа такси

# О чем приложение

Приложение для заказа такси. Полный набор работы с клиентом - личный кабинет, история поездок, промокоды, сохранение карт, тарифы и другое.

# Задача проекта

Сверстать по макету в Figma приложение без добавления бекенда. Главное условие - адаптивная гибкая верстка, одинаково хорошо подстраивающаяся под все разновидности экранов, - соблюдено. 

# Выполненность

Все страницы приложения (60 штук) были сверстаны за 14 дней. 
Проверка качества верстки выполняется так: запускаются на 6-7 устройствах с различными экранами - меняется диагональ, разрешение, наличие/отсутсвие кнопок и вид челки. 
Чтобы заказчик мог убедиться в качестве верстки самостоятельно:

* собираю приложение - создаю apk-файл, который можно запустить на своем телефоне
* наполняю все страницы и списки тестовыми данными - можно увидеть, как в будущем будут смотреться реальные данные пользователя

# Технологии

* Material Components
* Адаптивная XML-верстка
* Masked EditText - поля ввода с маской (телефон, email, даты)
* AndResGuard - для абфускации ресурсов и кода (безопасность и уменьшение веса приложения)
* Кастомные View - через создание drawable-ресурсов и переопределение View-классов
* Navigation Menu - боковое меню
