Аудиокниги для компании "1С"

О заказчике

Приложение разрабатывалось для компании 1C-Publishing LLC

Да, "1С", как оказалось, занимается не только бухгалтерией, но и играми, электронными книгами, аудиокнигами...

Моя область ответственности

Полный цикл разработки Android-приложения:

  • архитектура кода - масштабируемость, простота обновления кодовой базы
  • выпуск новых функций - новые способы оплаты, редизайны, улучшение поиска и скорости загрузки приложения
  • исправление багов

Приложение опубликовано в Play Market:

1С:Аудиоклуб – библиотека аудиокниг 1С

Команда

Наша команда состяла из бекендщика, менеджера и дизайнера. Хорошие люди, которых с радостью позову в новые проекты.

Интересные технические задача

  • внедрение оплаты через ЮKassa - отказ от зарубежных аналогов в связи с санкциями
  • упрощение API - по моим рекомендациям перерабатывалось клиент-серверное общение. Это было необходимо для ускорения загрузки контента в приложении
  • динамические фильтры - поиск по книгам стал удобнее благодаря гибкой системе выбора параметров
  • полный переход на асинхронный код и пагинацию для списков

Технологии

  • Jetpack Compose
  • Kotlin Coroutines
  • Build Flavors для A\B тестирования
  • ExoPlayer - воспроизведение аудио
  • Room - локальная БД
  • Hilt - DI фреймворк
  • YooKassa\Google Billing - оплата книг
  • WorkManager - упорядочивание фоновых задач
  • Retrofit 2 - генерация API-запросов
  • Firebase Analytic + Crashlytics
  • Firebase Auth + Firestore
  • Glide - загрузка картинок
  • Navigation Components - наглядная карта приложения
